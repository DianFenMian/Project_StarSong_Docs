# 服务器插件指南

## 基础功能
- **传送/领地/经济**：按 `Shift+F` 打开菜单
- ![鞘翅刷新机制](./pic/ElytraVault.png)
- **鞘翅刷新**：使用末地水晶激活末地船中的鞘翅宝库，让每位玩家都有公平获取机会

## 文字与装扮

### 多彩文字
- 使用 `&` 符号代替 `§` 输入彩色文字
- 支持 RGB 彩色文本，格式：`&#66ccff文字内容`
- 可在聊天、命名功能中使用
- 测试命令：`/echo [文本内容]`（仅自己可见）

### 更改聊天前缀/后缀/昵称
| 功能 | 命令 | 费用 |
|------|------|------|
| 设置前缀 | `/u chat prefix set [前缀]` | 198星点+100经验 |
| 重置前缀 | `/u chat prefix remove` | 免费 |
| 设置后缀 | `/u chat suffix set [后缀]` | 99星点+100经验 |
| 重置后缀 | `/u chat suffix remove` | 免费 |
| 自定义昵称 | `/nick [昵称]` | 免费 |
| 清除昵称 | `/nick off` | 免费 |

## 物品相关功能

### 邮箱系统
| 功能 | 命令 | 费用 |
|------|------|------|
| 设置邮箱 | `/u mail box set` | 10星点 |
| 取消邮箱 | `/u mail box clear` | 免费 |
| 查看邮箱位置 | `/u mail box info` | 免费 |
| 发送物品 | `/u mail sendto <玩家ID>` | 1星点/次(潜影盒18星点) |

- 手持物品发送邮件
- 支持箱子、木桶等容器作为邮箱

### 物品展示
- **展示手中物品**：`/u show`（全服可见）
- **展示框保护**：
  1. 放置展示框并放入物品
  2. 执行 `/u lock setup`
  3. 右键查看物品信息，Shift+右键旋转
  4. 取消保护：`/u lock remove`

### 物品重命名
- 命令：`/u item rename [名称]`
- 费用：10星点/个（一组64个需640星点）
- 支持多彩文字和空格

## 战斗与装备

### 经验胶囊
- 存储经验：手持附魔之瓶，执行 `/u xp store [经验值]`
- 提取经验：
  - `/u xp take [经验值]`
  - 非潜行状态双击使用键（默认提取1%或10点）
  - `/u xp quicktake [经验值]` 自定义提取量
  - 潜行状态使用键释放全部经验

### 自动飞行
- 手持刷子进入鞘翅飞行状态
- 速度过低时自动燃放烟花维持飞行
- 按住潜行键停止自动燃放

## 其他实用功能

### 编辑告示牌
- 命令：`/u signedit [行数1-4] [内容]`
- 支持多彩文字和空格
- 不可编辑特殊功能牌子（如商店牌）

### 席地而坐
- 主手为空，右键方块顶部
- 命令 `/u sit` 开关此功能
- 支持毛毯、床（白天）、台阶、楼梯等

### 发红包
- 命令：`/u redbag create [fixed|lucky] [金额] [数目] [口令]`
- fixed：固定金额红包
- lucky：拼手气红包（总金额）

### 离线消息
- 离线或挂机时的消息会暂存
- 上线后使用 `/u news` 查看未读消息
- 支持商城、邮箱、商店消息
